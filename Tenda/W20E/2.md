### description
A buffer overflow vulnerability was discovered in Tenda W20E V4.0br_V15.11.0.6. Attackers can exploit this vulnerability by specifying the values of `wifiFilterListDeviceMAC` and `wifiFilterListRemark`. When these values are passed to `modifyWifiFilterRules` and formatted into the fixed-size `sMibValue` buffer by `sprintf` without length validation, it may cause a buffer overflow.
