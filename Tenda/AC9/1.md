### description
A buffer overflow vulnerability was discovered in Tenda AC9 US_AC9V3.0RTL_V15.03.06.42_multi_TD01.bin Attackers can exploit this vulnerability by specifying the value of `timeZone`. When `timeZone` is passed to `form_fast_setting_wifi_set` and written into fixed-size stack buffers through `sscanf` without length validation, it may cause a buffer overflow.
