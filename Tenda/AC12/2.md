### description
A buffer overflow vulnerability was discovered in Tenda AC12 US_AC1206V1.0RTL_V15.03.06.23_multi_TD01.bin. Attackers can exploit this vulnerability by specifying the value of `shareSpeed`. When `shareSpeed` is passed from `fromSetWifiGusetBasic` through `set_wl_guest_qos_list` to `set_wl_guest_iplist` and formatted into fixed-size stack buffers by `sprintf` without length validation, it may cause a buffer overflow.

### code
