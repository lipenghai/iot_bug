### description
A buffer overflow vulnerability was discovered in Tenda AC12 US_AC1206V1.0RTL_V15.03.06.23_multi_TD01.bin. Attackers can exploit this vulnerability by specifying the value of `devName`. When `devName` is passed from `formSetDeviceName` to `set_device_name` and formatted into the fixed-size `mib_vlaue` buffer by `sprintf` without length validation, it may cause a buffer overflow.
