### description
A buffer overflow vulnerability was discovered in Tenda AC12 US_AC1206V1.0RTL_V15.03.06.23_multi_TD01.bin. Attackers can exploit this vulnerability by specifying the value of `rebootTime`. When `rebootTime` is passed to `formSetRebootTimer`, parsed by `sscanf`, and used to format the fixed-size `end_time` buffer through `sprintf` without proper bounds validation, it may cause a buffer overflow.
